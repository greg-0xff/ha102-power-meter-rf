# Contributing

Contributions are welcome, especially:

- Raw RF captures (hex or bit-level) from Mieo HA102 meters
- Confirmation of field meanings on other hardware revisions
- Improvements to rtl_433 parameters or demod stability
- Documentation corrections (no guessing please)

## Submitting data
If you submit samples, include:
- Frequency and rtl_433 command used
- Whether CRC passed
- Raw frame data (before parsing)
- Notes on device state (load on/off, pairing mode, etc.)

## What not to do
- Please do not submit guesses without data
- Do not request support for unrelated devices without captures

This project documents observed behavior only.
